What Happens When You Run This Command:
Mininet will create a virtual network with a linear topology of 3 hosts, where each host is connected to an OVS switch.
The switches will use OpenFlow 1.3 to communicate with the remote SDN controller at 192.168.100.233.
The remote controller can manage and control the flow tables of the switches using OpenFlow.
