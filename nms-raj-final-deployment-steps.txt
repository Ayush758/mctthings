** Creation of mctnms database user and password:
- sudo -u postgres createuser -P opennms

- sudo -u postgres createdb -O opennms opennms
** Set a password for Postgresql superuser:
(Note: Please enter your own password in 'Admin@123')
- sudo -u postgres psql -c "ALTER USER postgres WITH PASSWORD 'Admin@123';"
